PROJECTS

#1

BLUETOOTH CONTROLLED BOT

The Bluetooth Controlled Bot is a mobile robot designed for wireless control using a smartphone via the HC-05 Bluetooth module. Powered by an Arduino Uno and an L293D motor driver, the bot uses four motors and wheels mounted on a sturdy chassis. Two 18650 batteries housed in a battery holder supply the necessary power, while a switch allows easy on/off control. This project demonstrates basic robotics and wireless communication, making it ideal for beginners interested in embedded systems and remote control applications.

#2

LINE FOLLOWER BOT

The Line Follower Bot is an autonomous robot designed to detect and follow a path using an IR sensor array. It is built on an Arduino Nano microcontroller, which processes signals from the sensors to control movement. The L298N motor driver powers two N-20 motors connected to a pair of wheels, with a castor wheel added for balance and smooth turning. The bot is powered by two 18650 batteries placed in a battery holder, making it portable and efficient. This project is a hands-on introduction to robotics, sensor-based navigation, and basic embedded programming.

#1

DRONE

The Drone project is a work in progress aimed at building a quadcopter with stable flight control and sensor feedback integration. At its core is the powerful Teensy 4.0 microcontroller, paired with the MPU6050 for real-time motion tracking and the BMP280 for altitude sensing. The drone is powered by four 1400KV brushless DC motors controlled via electronic speed controllers (ESCs) and driven by an 11.1V 2200mAh LiPo battery. LEDs with resistors are included for visual indicators and orientation. This ongoing project focuses on achieving flight stability, sensor fusion, and future expansion into autonomous control and GPS navigation.
